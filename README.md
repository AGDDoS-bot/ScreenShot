# 自动截图

向<https://github.com/xiaozhu2007/ScreenShot/blob/master/index.js>按照格式提交PR

添加一行

```javascript
snap("https://<YOUR DOMAIN>", "./snap/<DOMAIN NAME>.png");
```

PR状态: **允许** 添加截图请求

默认大小1024\*655

示例
`https://cdn.jsdelivr.net/gh/xiaozhu2007/ScreenShot@master/snap/baidu.com.png`
